# Projeto-MVC-em-Delphi
Projeto VCL em Delphi, estruturado em MVC. Utilizei Clean Code, MVC e OO, trazendo tanto um exemplo de cadastro de usuário com DataModule, quanto outro cadastro de clientes sem a utilização de DataModule.

O Banco de Dados é o Firebird, mas nada impede a utilização de outro banco, sendo necessário somente alterar as configurações na classe de conexão do projeto.

Para abrir e compilar este projeto, sugiro que seja a partir do Delphi 2010, apesar de eu ter utilizado componentes nativos da linguagem.
